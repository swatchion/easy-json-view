## 🏗️ 技术架构

### 最终技术栈
```
前端框架: Dioxus 0.6.3 (Rust GUI框架)
编译目标: WebAssembly (wasm32-unknown-unknown)
样式系统: Tailwind CSS 4
数据存储: localStorage (浏览器本地存储)
构建工具: Cargo + wasm-bindgen + wasm-opt
开发服务: Python HTTP Server
测试框架: Rust内置测试 + Criterion基准测试
```

### 架构调整说明
- **平台**: 桌面版 → Web版 (解决GTK依赖问题)
- **存储**: SQLite → localStorage (Web平台适配)
- **部署**: AppImage → 静态Web应用 (简化部署)

## 🚀 使用方法

### 快速开始
```bash
# 1. 构建项目
./build.sh

# 2. 启动应用
cd dist && python3 serve.py

# 3. 访问应用
# 浏览器打开 http://localhost:8000
```

### 功能使用
1. **JSON输入**: 在下方文本框输入JSON内容
2. **选择缩进**: 顶部选择2/4/8空格缩进
3. **格式化**: 点击"格式化"或按Ctrl+Enter
4. **查看结果**: 上方显示格式化后的JSON
5. **历史管理**: 左侧搜索和管理历史记录
